Learn More About The Cloud Farm Project.

The Cloud Farm Project is the application of cloud computing and IoT to configure, monitor, and maintain automated greenhouses. The Cloud Farm Server connects to a client device (farm), handling its logs through the application. We enable users to access their clients from anywhere in the world, make changes to procedures, experiment for different results, and eventually talk and post about it. The Cloud Farm Application offers a simple platform for agricultural science, creativity, and the fun moments you get watching something grow! Have fun planting!
